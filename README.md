# QAZAQSTAN.travel — концепт-редизайн

Современный концепт-редизайн [kazakhstan.travel](https://kazakhstan.travel/ru) с интерактивной
3D-картой Казахстана и режимом «прогулки» по туристическим локациям.

**Демо:** https://stellasdeutsch-dev.github.io/kaz-travel/

## Что внутри

- **3D-карта страны** на Three.js — реальный контур границ, экструдированная плита со светящимся
  золотым краем, звёзды, объёмная пыль, bloom. Вращается мышью, авто-вращение в покое.
- **8 кликабельных локаций** — маркер открывает панель с описанием и кнопкой «Прогуляться в 3D».
- **8 процедурных сцен-прогулок** (генерируются кодом, без внешних моделей): Чарынский каньон,
  Большое Алматинское озеро, озеро Каинды, Астана ночью, Туркестан, Бозжыра, Бурабай, Байконур.
  Орбитальная камера + перемещение на `WASD`.
- Тёмная премиальная тема, стекло, крупная типографика, анимации появления, адаптив до мобильных.

## Стек

Vite + Three.js, без фреймворков. Вся 3D-геометрия и ландшафты генерируются процедурно
(шум Перлина, ExtrudeGeometry, InstancedMesh).

## Разработка

```bash
npm install
npm run dev      # http://localhost:5183
npm run build
```

## Структура

| Файл | Назначение |
| --- | --- |
| `src/app3d.js` | Рендерер, карта, маркеры, перелёты камеры, режим прогулки |
| `src/scenes.js` | Процедурные генераторы восьми локаций |
| `src/locations.js` | Данные локаций и текстов |
| `src/style.css` | Дизайн-система |
| `public/kaz-border.json` | Упрощённый контур границ (503 точки) |

Деплой — GitHub Actions при пуше в `main` (`.github/workflows/deploy.yml`).
